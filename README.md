# CaseBoom Pro v5

Версия с реферальной системой, ежедневным бонусом, уведомлениями, улучшенной рулеткой и расширенными батлами.

## Что добавлено в v5

- Реферальная система:
  - личный код;
  - ссылка `/r/CODE`;
  - бонус пригласившему;
  - история приглашений.
- Ежедневный бонус:
  - получение раз в сутки;
  - серия/streak;
  - начисление баланса;
  - уведомление.
- Уведомления:
  - страница пользователя;
  - mark all read;
  - уведомления за кейсы, daily, рефералов;
  - админская отправка уведомлений.
- Улучшенная рулетка открытия:
  - лента предметов из конкретного кейса;
  - центральный индикатор;
  - карточки с картинками.
- Батлы:
  - страница комнаты;
  - вход в комнату;
  - участники.
- Всё из Pro v4:
  - готовность к бесплатному деплою Vercel + Neon;
  - Steam-вход;
  - кейсы, скины, апгрейд, контракты;
  - админка, промокоды, авто-закупка, трейды.

## Запуск локально

```bash
npm install
cp .env.example .env
docker compose up -d
npx prisma migrate dev --name init
npm run prisma:seed
npm run dev
```

## Запуск в интернете

Смотри:

```text
DEPLOY_FREE_VERCEL_NEON_RU.md
```

## Важно

Для реальных платежей, реальных скинов, Steam Trade Bot и закупки в маркетах нужны юридическая проверка, договоры с провайдерами и соблюдение правил Steam/Valve.
