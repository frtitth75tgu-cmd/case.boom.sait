# CaseBoom Ultimate v25 — Candy Build

Полный проект, не патч.

## Главное
- Фиолетово-неоновый стиль.
- Boom Core логотип.
- Отдельного каталога скинов нет.
- Больше страниц и разделов.
- Расширенная админка.
- Документация.
- Public-ассеты-заготовки.
- Steam не ломали, база взята со стабильной версии.

## Установка

```bash
npm install
cp .env.example .env
npx prisma db push
npm run dev
```

Старый проект не смешивай. Распакуй как отдельную папку.
